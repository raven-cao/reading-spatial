Gene Expression Vector

The mathematical representation of a cell's state

Table vs. vector: same thing, different use

A spreadsheet row — 6,000 gene count columns for one cell — is a vector. The table is for human reading; the vector is for computation. The distinction matters because "vector" imports a whole mathematical framework: addition, scaling, distance, angle.

Why "ordered" matters

Position encodes meaning. [23, 12, 0] ≠ [0, 12, 23] if the first slot is CD8A and the third is EPCAM. The ordering is arbitrary but must be consistent across all cells. Unlike a set, a vector cares about position.

Cosine similarity

cos(A, B) = (A · B) / (|A| × |B|)

Dot product divided by the product of lengths. This cancels out magnitude — a cell with twice the sequencing depth but the same expression pattern gets similarity 1.0 to the original. Only the direction (pattern of which genes are high/low) is compared, not absolute counts. This is biologically appropriate: technical differences in sequencing depth don't reflect biological state differences.

The "direction" of a cell state
In a 2-gene toy world: an effector T cell points toward (GZMB-high, PD-1-low). An exhausted T cell points toward (GZMB-low, PD-1-high). These are different directions in expression space. In 6,000 dimensions you can't visualize the direction, but the math works exactly the same way. "Direction" = the pattern of which genes are relatively high and which are low.

Why high-dimensional spaces are valid even if unimaginable

The rules (distance, angle, addition) are defined axiomatically — they extend to any number of dimensions by construction, not by physical analogy. The reason we trust 6,000-dimensional vectors to capture biology: predictions made in that space (cluster this cell with those, these two cells are similar) consistently match experimental validation. The math works; imagination is optional.

Why vectors instead of other similarity tools?

Other measures exist (Manhattan distance, Jaccard similarity). Vectors were chosen because cosine similarity captures pattern alignment — two cells in the same biological state but different sequencing depths. Also, vectors integrate into the larger ecosystem of linear algebra that neural network operations (dot products, matrix multiplications) are built on. The tool fits the downstream computation.
